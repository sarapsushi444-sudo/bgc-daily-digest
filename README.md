# BGC Daily Digest
Daily tech, crypto, markets & Philippines news digest — powered by OpenClaw.

🌐 **Live site:** https://sarapsushi444-sudo.github.io/bgc-daily-digest

## Topics
- 🧠 AI / LLM News & Trends
- ₿ Bitcoin & Crypto
- 🛢️ Oil, Gas & Geopolitics
- 📈 US Markets & Economics
- 🇵🇭 Philippines / Metro Manila

## How it works
- OpenClaw runs tech-news-digest daily at 5PM PHT
- Output is pushed to this GitHub Pages site
- Build script converts markdown archives to HTML pages

## Adding/editing sources
See `config/tech-news-digest-sources.json` in the OpenClaw workspace.
